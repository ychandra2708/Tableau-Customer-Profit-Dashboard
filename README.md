# 📊 Customer & Profit Analytics Dashboard

## 📌 Overview
This project presents an interactive Tableau dashboard that analyzes customer behavior and profitability.

---

## 🎯 Key Features
- Total Customers
- Average Profit per Order
- Profit Margin
- Total Sales

- Top Customers
- Monthly Profit Trend
- Profit by Region
- Profit by Category
- Profit by Segment

---

## 🛠️ Tools Used
- Tableau
- Excel
